# Latest Wallpaper
![](https://www.bing.com/th?id=OHR.SantaGlobe_DE-DE3146908571_UHD.jpg)
[2025-12-25 Kleine Welten, großer Zauber Schneekugeln auf einem Weihnachtsmarkt in Heidelberg, Baden-Württemberg(© George Pachantouris/Getty Images)](https://www.bing.com/th?id=OHR.SantaGlobe_DE-DE3146908571_UHD.jpg)
Ein sanftes Schütteln genügt – und schon tanzt Schnee um eine kleine Welt im Glas. Schneekugeln haben eine faszinierende Geschichte: Erste wassergefüllte Glaskugeln gab es bereits im 16. Jahrhundert, doch die Schneekugel, wie wir sie heute kennen, erfand Erwin Perzy um 1900 in Wien. Sein Modell der Basilika von Mariazell, gefüllt mit Wasser und feinem Grieß als Schnee, wurde zum Kultobjekt und machte die Schneekugel weltweit bekannt. Heute erstrahlen diese Miniaturwelten in unzähligen Designs und liebevollen Motiven, wie das heutige Bild vom Heidelberger Weihnachtsmarkt eindrucksvoll zeigt.
