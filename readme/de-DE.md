Was wie eine Filmkulisse wirkt, ist ein historischer Ort: Ksar Ouled Soltane liegt im Süden Tunesiens im Gouvernement Tataouine, nahe der Sahara. Die mehrstöckigen Speicherburgen aus Lehm wurden im 15. Jahrhundert von Berbergruppen als sichere Vorratslager errichtet. Sie schützten das Getreide vor Hitze, Diebstahl und Verderb. Mit ihrer klaren, funktionalen Architektur prägen sie bis heute die trockene Landschaft der Region.
