![](https://www.bing.com/th?id=OHR.MonaValePool_DE-DE2862421756_UHD.jpg)
[2025-05-27 Eine Badewanne im Meer Mona Vale Ozean-Pool, Sydney, Australien(© jamenpercy/Getty Images)](https://www.bing.com/th?id=OHR.MonaValePool_DE-DE2862421756_UHD.jpg)
Entspannt in einer großzügigen Meerwasser-Badewanne baden, umgeben von den tosenden Wellen des Ozeans? Dieser Traum wird im Mona Vale Ozean-Pool in Sydney Wirklichkeit. Das auf unserem heutigen Bild zu sehende Gezeitenbecken ist ein echtes Juwel an Australiens Küste. Malerisch zwischen den Stränden The Basin Beach und Mona Vale Beach gelegen, wirkt dieser Felsenpool fast wie ein Naturwunder. Er wurde in den 1930er Jahren während der Weltwirtschaftskrise aus einer natürlichen Felsplatte herausgearbeitet, als Teil eines Arbeitsbeschaffungsprogramms. Heute ist der über 30 Meter lange Pool modernisiert und wird von einem kleineren Kinderbecken ergänzt.
