# Latest Wallpaper
![](https://www.bing.com/th?id=OHR.CastleBlossoms_DE-DE0573709826_UHD.jpg)
[2026-04-06 Beständig durch die Zeit Burg Hirosaki mit Kirschblüten, Hirosaki, Japan(© Glenn Waters/Getty Images)](https://www.bing.com/th?id=OHR.CastleBlossoms_DE-DE0573709826_UHD.jpg)
Ein Blick nach Nordjapan um das Jahr 1600 zeigt eine Welt, in der Burgen das Zentrum des öffentlichen Lebens bildeten. Sie dienten als Verwaltungszentren, schützten Handelswege und waren fest in den Alltag eingebunden. Daran knüpft der japanische Burgentag an, der jedes Jahr am 6. April begangen wird. Der Hintergrund ist ein Wortspiel: Vier und sechs heißen im Japanischen „shi“ und „ro“ – zusammen ergeben sie „shiro“, das Wort für Burg.
