Es gibt Wege, die man nicht so schnell vergisst. Entlang der abgelegenen Südwestküste von Vancouver Island verläuft der West Coast Trail, einer der bekanntesten Wanderpfade Kanadas. Die rund 75 Kilometer lange Route führt durch Regenwald, an Stränden entlang und vorbei an zerklüfteten Küstenabschnitten im Pacific‑Rim‑Nationalpark. Der Weg wurde im Jahr 1907 als Rettungsroute für Schiffbrüchige im berüchtigten „Graveyard of the Pacific“ angelegt.
