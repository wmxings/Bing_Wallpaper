# Latest Wallpaper
![](https://www.bing.com/th?id=OHR.YosemiteWinter_DE-DE7894807438_UHD.jpg)
[2025-12-13 Frostige Schönheit Merced River, Yosemite-Nationalpark, Kalifornien, USA(© Ron and Patty Thomas/Getty Images)](https://www.bing.com/th?id=OHR.YosemiteWinter_DE-DE7894807438_UHD.jpg)
Mitten im Winter verwandelt sich der Yosemite-Nationalpark in Kalifornien, USA, in eine majestätische, schneebedeckte Landschaft. Auf dem heutigen Bild ist der Merced River zu sehen, dessen Name auf Spanisch „Barmherzigkeit“ bedeutet. Er entspringt in den hohen Gipfeln der Sierra Nevada und durchfließt von Gletschern geformte Täler, die von imposanten Granitwänden umgeben sind. Der Fluss ist Teil der geologischen Geschichte des Parks und speist einzigartige Ökosysteme. In den kältesten Monaten formt das Eis originelle Gebilde an seinen Ufern und bietet ein Naturschauspiel, das seit Generationen Künstler und Fotografen inspiriert.
