[2026-02-06 Die Dolomiten im Olympiafieber Gipfel Ra Gusela am Giau-Pass nahe Cortina d’Ampezzo, Italien(© Tomasz Podolski/Getty Images)](https://www.bing.com/th?id=OHR.GiauPass_DE-DE5864987336_UHD.jpg)
Mit dem Start der Olympischen Winterspiele 2026 richtet sich die Aufmerksamkeit auf ein Panorama, das wie geschaffen scheint für olympische Momente: die Dolomiten in Norditalien. Über dem Giau‑Pass erhebt sich der rund 2.600 Meter hohe Ra Gusela, dessen markante Felsformation die alpine Kulisse der Spiele eindrucksvoll widerspiegelt. Vom 6. bis 22. Februar treten Athletinnen und Athleten aus aller Welt in Italien gegeneinander an. Die Winterspiele werden dabei an zwei Gastgeberorten ausgetragen. Mailand ist Austragungsort der Eissportarten, während die alpinen Wettbewerbe rund um Cortina d’Ampezzo stattfinden. Dort feiert in diesem Jahr auch das Skibergsteigen seine olympische Premiere, eine Disziplin, die in hochalpinen Regionen seit Langem an Bedeutung gewinnt.
