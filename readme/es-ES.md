¿Qué desearías si una sola linterna pudiera llevar tu deseo al cielo? Esta noche, en el norte de Tailandia, el festival Yi Peng te brinda esa oportunidad. Esta tradición centenaria del antiguo reino de Lanna celebra la luna llena del duodécimo mes lunar con miles de linternas de papel, las “khom loi”, que se elevan para honrar a Buda. Se cree que cada una se lleva la mala suerte y abre paso a un futuro más luminoso.
