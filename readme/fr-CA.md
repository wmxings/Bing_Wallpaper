Une touche de jaune vif parmi le feuillage? C'est sans doute une paruline du Canada. Chaque printemps, ces petits chanteurs illuminent le paysage canadien avant de rester tout l’été. Friands de chenilles et de coléoptères, ils évoluent dans les milieux humides, les bois et les jardins, nichant dans les arbustes et petits arbres.
