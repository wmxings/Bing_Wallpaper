À Olinda, au Brésil, la couleur est une promesse de bonheur. Sur les collines du Pernambouc, cette ville fondée au XVIᵉ siècle s’épanouit entre histoire et créativité. Les façades pastel côtoient les églises baroques, les ruelles pavées invitent à flâner, comme si le temps s’était arrêté. Ici, l’architecture coloniale dialogue avec les traditions afro-brésiliennes, tissant une identité vive.
