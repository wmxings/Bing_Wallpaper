Dans le Northumberland, au nord de l’Angleterre, un écureuil roux d’Eurasie s’agrippe à une branche givrée, serrant un cône de pin entre ses pattes. Sa fourrure ardente tranche sur la neige immobile, comme une étincelle dans le silence glacé. Infatigable, il amasse noisettes, graines et champignons, qu’il cache sous la mousse pour affronter les jours les plus rudes. Ces caches oubliées nourriront demain la forêt, faisant de lui un jardinier discret.
