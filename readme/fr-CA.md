À Nuuk, au Groenland, l’hiver enveloppe la ville d’un calme presque sacré. Les maisons colorées semblent veiller les unes sur les autres, comme dans certains villages nordiques du Canada, où la neige est autant un défi qu’une compagne familière. Ici, chaque déplacement offre un tableau mouvant : plaques de glace dérivantes, silhouettes de montagnes sculptées par les vents, et parfois, le frémissement d’une aurore qui effleure le ciel tel un voile en suspendu.
