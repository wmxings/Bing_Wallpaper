Sous un ciel clair, à Long Island aux États-Unis, un phoque gris repose sur un rocher, la nageoire levée vers l'horizon. Pour lui, c'est une question de chaleur, mais pour nous, c'est un signe : un salut qui traverse l'océan. Née en 1973 pour promouvoir la paix, la Journée mondiale du bonjour nous invite à renouer avec ce geste élémentaire célébré aujourd’hui dans plus de 180 pays.
