Blottie dans un lit de varech, la loutre de mer semble flotter entre deux mondes. Dans les eaux froides de l’Alaska, au cœur d’un refuge faunique maritime, elle se laisse porter, les pattes jointes, paisible. Cette image attendrissante cache une espèce au rôle essentiel : en se nourrissant d’oursins, les loutres protègent les forêts de kelp, véritables forêts sous-marines riches en biodiversité.
