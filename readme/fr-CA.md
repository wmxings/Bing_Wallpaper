Atteindre le sommet du Brocken, en Allemagne, c’est comme entrer dans un décor mouvant de nuages et de lumière. Le soleil dans le dos, la brume tout autour, une silhouette géante apparaît soudain, entourée d’un halo : le spectre du Brocken. Longtemps redoutée des randonneurs et alpinistes, cette illusion d’optique naît quand l’ombre humaine se projette sur le brouillard.
