Sur les rivages de Lüderitz, en Namibie, les flamants roses forment une ligne paisible entre mer et ciel. Ces oiseaux, les plus grands de leur espèce, se dressent jusqu’à 1,50 m, appuyés sur une seule patte. Leur plumage rose est le reflet de leur régime : algues, crustacés et pigments naturels. Plus ils mangent, plus ils s’illuminent.
