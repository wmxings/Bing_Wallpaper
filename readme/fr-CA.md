Sur la péninsule de Skagi, au nord de l’Islande, la crique de Kálfshamarsvík offre un spectacle fascinant : d’imposantes colonnes de basalte se dressent face aux vagues, taillées par le refroidissement lent d’une coulée de lave vieille de 2 millions d’années. Leurs formes hexagonales quasi parfaites semblent issues d’un monde mythique.
