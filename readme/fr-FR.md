Sous le ciel d’hiver d’Heidelberg, en Allemagne, les lumières des chalets se reflètent sur des rangées de boules à neige. Chacune renferme un univers miniature : un Père Noël figé dans sa ronde, une forêt étincelante, une promesse d’enfance. Dans cette ville romantique bordée par le Neckar, le marché de Noël emplit la vieille place de son parfum d’épices et de bois chaud.
