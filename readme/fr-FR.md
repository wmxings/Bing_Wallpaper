Les lamantins donnent l’impression de flotter au ralenti, presque assoupis. Pourtant, sous la surface, ils révèlent d’étonnantes particularités. Ce sont parmi les rares mammifères à renouveler leurs dents en continu : leurs molaires usées par les herbiers sableux avancent peu à peu, remplacées comme sur un tapis roulant - un atout pour un animal qui passe jusqu’à huit heures par jour à brouter.
