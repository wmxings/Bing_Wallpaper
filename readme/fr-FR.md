À l’extrême nord de la Suède, le soleil renaît timidement entre les troncs givrés. Ce matin de décembre, la lumière semble hésiter, caressant la neige d’un or pâle. C’est le solstice d’hiver, l’instant où la Terre incline le plus loin son regard du Soleil. La nuit atteint son règne absolu, mais déjà, imperceptiblement, les jours rallongent.
