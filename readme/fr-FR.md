Sous le soleil couchant, les chevaux de Camargue avancent, libres et puissants, dans les marais salants d'Aigues-Mortes. Ici, la nature ne se montre pas, elle se révèle brute, sauvage et indomptée. Emblèmes d'une région façonnée par l'eau et le sel, ils sont les descendants d'une lignée millénaire et sont habitués aux terres mouvantes, entre mer et étangs. Leurs sabots foulent les sols tremblants où poussent salicornes et tamaris, dans un ballet silencieux troublé seulement par les flamants roses.
