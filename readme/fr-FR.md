Les Cinque Torri se dressent comme cinq sentinelles minérales au cœur des Dolomites italiennes. Sculptées par des millions d’années d’érosion, ces tours de pierre offrent un décor à couper le souffle, entre prairies alpines et falaises vertigineuses. Mais ce paysage grandiose cache aussi une mémoire plus sombre : pendant la Première Guerre mondiale, ces sommets furent le théâtre de combats acharnés entre soldats italiens et austro-hongrois.
