Chaque printemps, la vallée de Skagit, au nord-ouest des États-Unis, se transforme en une mosaïque éclatante lorsque les premières tulipes percent la lumière d’avril. Pendant tout le mois, les visiteurs flânent parmi des jardins soigneusement dessinés ou longent les immenses champs où les couleurs s’étirent jusqu’aux montagnes encore poudrées de neige.
