Sous la neige qui danse, le renne avance, silhouette grise entre les troncs blancs de bouleaux. En Laponie, ils sont plus nombreux que les habitants, gardiens d’un territoire où la lumière se fait rare en hiver. Leurs sabots, ingénieusement adaptés, se resserrent pour mieux accrocher la glace, puis s’élargissent aux beaux jours, comme une chaussure qui change de saison.
