Pagayez sur une eau cristalline et laissez-vous surprendre par l’ombre imposante d’un lamantin glissant sous votre kayak. Bienvenue à Crystal River, en Floride, où l’on peut observer ces géants paisibles de près ! Chaque hiver, des centaines de lamantins fuient les eaux froides du golfe du Mexique pour se réfugier dans les sources chaudes, où l’eau reste à 22 °C. Inoffensifs, ces « vaches de mer » passent leurs journées à brouter les herbiers et à somnoler.
