Dall’alto, la superficie liscia del lago contrasta nettamente con il terreno accidentato che lo circonda. Il lago craterico di Hnausapollur, noto anche come Bláhylur, si trova nel cuore della Fjallabak Nature Reserve, un’area remota degli altopiani islandesi modellata più dall’attività vulcanica che dagli insediamenti umani. Il cratere si formò quando il magma incontrò le acque sotterranee, provocando un’esplosione che scavò il bacino circolare oggi visibile.
