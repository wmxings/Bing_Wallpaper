ここは、パパガヨビーチ。スペインのカナリア諸島に属するランサローテ島南部に位置します。ランサローテ島は長い火山活動の歴史を持ち、特に 18 世紀の噴火が現在の地形形成に大きく影響しました。そしてパパガヨビーチは、火山岩の断崖に囲まれており、それが防波堤になることで、波が和らげられ、ビーチには穏やかで透明度の高い海が広がります。黄金色の砂浜は、砕けた貝殻と火山岩が混ざり合ってできたもので、この島の成り立ちを感じさせます。
