ここは、米国オレゴン州にあるスミスロック州立公園。度重なる火山噴火によって形成された約 41km × 27km の巨大なクルックド・リバー・カルデラ内に位置しています。クルックド川によって削られた峡谷の周囲には、火山灰からなる急峻な溶結凝灰岩の断崖が広がっています。また、イヌワシやカワウソ、ピューマなど多様な野生生物が生息しており、オレゴン州の代表的な観光名所「七つの自然の驚異」の一つとして知られています。
