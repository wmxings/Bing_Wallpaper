メガネグマは、南米のアンデス山脈に分布するクマ科の動物で、クマ科では南米に生息する唯一の種です。目の周囲にある淡い模様が眼鏡のように見えることから、この名前が付けられました。非常に木登りが得意で、樹上に巣を作ることもあります。主に果実を食べますが、時には昆虫や小型の哺乳類なども捕食します。生息地の減少や狩猟、人間活動の影響により、国際自然保護連合（IUCN）では、「危急種（Vulnerable）」に分類されているため、その保護が重要視されています。
