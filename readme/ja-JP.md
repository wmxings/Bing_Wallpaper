ここは、イタリア・リグーリア海岸の港町ヴェルナッツァ。世界遺産である 5 つの村「チンクエ・テッレ」の一つであり、唯一天然港を中心に築かれた村です。1080年に要塞集落として初めて記録に登場しており、その後はジェノヴァ共和国の沿岸防衛拠点として栄えました。湾を見下ろすドーリア城跡や見張り塔がその面影を今に伝えており、海辺にはサンタ・マルゲリータ教会が佇み、鐘楼が港を見守ります。急斜面には石積みの段々畑が広がり、ブドウやオリーブが育まれています。色彩豊かな家並みと港を眺めながら、フォカッチャを味わえば、絵葉書のようなひとときを過ごせます。
