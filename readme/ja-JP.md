今日の画像は、スペインのマヨルカ島にあるフォルメントール灯台。バレアレス諸島最高峰の灯台で、フォルメントール岬の先端にある断崖の上、海抜 210m に位置しています。1863 年に建設され、同年に運用が開始されました。周囲の断崖は、隆起した石灰岩が風や波による侵食を受けて形づくられたもので、この地域独特の荒涼とした美しさを生み出しています。
