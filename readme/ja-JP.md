今日の画像は、米国フロリダ州の湿地で羽繕いをするユキコサギ。雪のように純白の羽と、黒い脚に鮮やかな黄色の足指が印象的な鳥です。羽毛を清潔にし、防水性を保つために行う羽繕いは、浅瀬で小魚や甲殻類、昆虫などを捕食するユキコサギにとって欠かせない行動です。繁殖期に現れる綺麗な繁殖羽を目的とした狩猟により、一時は絶滅寸前まで追い込まれましたが、保護活動によって個体数は少しずつ回復し、現在では一年を通して湿地でよく見られる存在となっています。
