モクレンが咲き誇る季節がやってきました。「モクレン（木蓮）」という名前は、花の形が蓮華に似ており、香りが蘭に似ていることから名付けられたもので、空に向かって花を一斉に咲かせる姿が特徴です。一般的に「モクレン」と呼ぶ場合は、紫色のシモクレンを指すことが多く、白い花のハクモクレン（白木蓮）とは別種のモクレンです。中国南部原産ですが、日本には古くから導入されており、今も昔も、庭園や公園で親しまれることの多い花です。
