[2026-06-16 今日は世界ウミガメの日 優雅に泳ぐタイマイ, ケイマン諸島(© Alex Mustard/Nature Picture Library)](https://www.bing.com/th?id=OHR.SevenMileTurtle_JA-JP9577560353_UHD.jpg)
今日 6 月 16 日は「世界ウミガメの日」。ウミガメの存在やその保護の大切さについて、改めて考える日です。世界に生息している 7 種のウミガメのうち、日本の砂浜で産卵が確認されているのは、アカウミガメ、アオウミガメ、そして画像のタイマイの 3 種です。タイマイは、模様のある甲羅と、タカのくちばしのように先のとがった口元が特徴です。沿岸の浅い海域を好み、主に海綿動物を食べることで、サンゴ礁のバランスを保つ役割を果たしています。現在、絶滅の危機に瀕しており、ウミガメの中でも特に希少な種のひとつです。
