あじさい（紫陽花）は、梅雨の時期を代表する美しい花です。今日の画像は、岩手県奥州市で撮影されたあじさい。例年、6 月下旬から 7 月下旬ごろにかけて見頃を迎えます。あじさいは、画像のような青以外にも様々な色がありますが、これは土壌の酸性度によって変化するものです。日本は、弱酸性の土壌が広く分布しているため、青や紫の花が多く見られます。梅雨の曇り空の下に広がる爽やかな青いあじさいが、ジメジメとした梅雨の空気を和らげてくれます。
