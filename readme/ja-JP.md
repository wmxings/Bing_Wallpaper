米国アリゾナ州に位置するグランドキャニオン国立公園は、1919 年の今日、2 月 26 日に国立公園に指定されました。この公園は、コロラド川が数百万年にわたって削り出した壮大な峡谷で形成されており、299 種以上の野鳥や 74 種の哺乳動物など、多種多様な動物が多く生息しています。また、世界で最も多くの化石が発見された場所でもあり、1979 年には世界遺産に登録されました。
