[2026-05-21 今日は国際お茶の日 四川省の茶畑, 中国(© lzf/Shutterstock)](https://www.bing.com/th?id=OHR.SichuanTea_JA-JP8777846900_UHD.jpg)
今日 5 月 21 日は国際連合が定める「国際お茶の日」。そして画像は、中国・四川省の段々畑の茶畑です。この地域は茶文化の発祥地の一つとされており、何世紀にもわたってお茶が栽培されてきました。また、近隣の蒙山は約 2000 年前、西漢の時代に甘露寺の普慧禅師（呉理真）が茶樹を植えて、世界で初めて体系的な茶の栽培を行った地ともいわれています。
