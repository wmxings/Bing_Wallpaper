今日、4 月 17 日は「国際コウモリ感謝の日」です。日本ではあまりなじみがない記念日ですが、果物や昆虫を食べて生態系を支えているコウモリへの理解を深め、保全への意識を高めることを目的としています。そして画像は、オーストラリアで撮影された、赤ちゃんを抱えるハイイロオオコウモリ。オーストラリア最大級のコウモリで、翼は広げると 1m 以上にもなります。また、コウモリが使うことで知られる反響定位（エコーロケーション）は使用しない種であり、鋭い視覚と嗅覚を使って飛びます。
