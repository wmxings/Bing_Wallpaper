今日 3 月 27 日は「さくらの日」。日本さくらの会が 1992 年に制定した桜の記念日で、七十二候の「桜始開」の頃にもあたり、暦の上では桜が咲き始める時期とされています。そして、画像の桜並木は、早咲きの桜として知られるカンヒザクラが咲く沖縄で撮影されました。沖縄では、本州とは逆に桜前線が北から南へと南下するのが特徴で、本島北部では、1 月から 2 月上旬にかけて見頃を迎えます。
