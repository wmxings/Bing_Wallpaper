[2026-02-23 今日は「富士山の日」 富士山, 静岡県(© Cravetiger/Getty Images)](https://www.bing.com/th?id=OHR.MountFuji2026_JA-JP8594170652_UHD.jpg)
今日 2 月 23 日は天皇誕生日であり、「富士山 (223) 」の語呂合わせで、静岡県と山梨県が制定した「富士山の日」でもあります。この日を祝して、静岡県の景勝地、日本平から望む富士山と清水港の夕景をお届けします。清水港は、長崎港と神戸港と並び「日本三大美港」に数えられており、クルーズ客船の寄港も多く、近年は貿易港としてだけではなく、観光港としても整備されています。
