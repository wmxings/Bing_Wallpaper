今日 5 月 3 日は国際ヒョウの日。絶滅の危機にあるヒョウの現状を広め、保護の重要性を訴えるために設けられた記念日です。ヒョウは、ネコ科で最も広域に分布する種であり、アフリカ大陸からアラビア半島・東南アジア・極東ロシアなどに生息しています。森林や山岳地帯、広大なサバンナなど多様な環境に高い適応力を持つものの、生息地の破壊や害獣駆除、毛皮目的の狩猟などにより、個体数は世界的に減少しています。
