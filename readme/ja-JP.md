今週は全米花粉媒介者週間。ミツバチをはじめ、蝶や鳥、コウモリ、甲虫といった花粉媒介者の存在に気づき、守ることを呼びかける期間です。画像のように、蝶が蜜を吸う際に、体に花粉が付着します。この花粉をつけた蝶が次の花へ移動することで、植物は受粉し、繁殖します。花粉媒介者は植物の繁殖や農作物の安定した供給に欠かせない存在です。生態系の保全や私たちの豊かな食生活を維持するためにも、彼らの保護は重要です。
