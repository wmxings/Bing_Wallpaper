Hoje celebramos o Dia Internacional das Montanhas, uma data dedicada a reconhecer sua importância para o nosso planeta. Elas não são apenas paisagens bonitas — são verdadeiras fontes de vida. As montanhas cobrem quase um quarto da superfície terrestre, abrigam 15% da população mundial e são refúgio para uma rica diversidade biológica, incluindo plantas, animais e microrganismos.
