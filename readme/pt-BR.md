No interior do Rio Grande do Sul, São Miguel das Missões é um lugar onde o passado parece respirar entre ruínas de pedra avermelhada. O que hoje encanta os visitantes são os vestígios de uma antiga aldeia construída entre os séculos XVII e XVIII, parte de um projeto de integração entre europeus e povos indígenas.
