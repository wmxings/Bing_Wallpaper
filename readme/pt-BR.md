Da superfície marcada por crateras da Lua, a Terra é um fino arco luminoso suspenso na escuridão. É uma inversão curiosa: aqui de cima, nosso planeta não é o cenário principal, mas uma presença distante, quase frágil. O relevo lunar, moldado por bilhões de anos de impactos, guarda uma espécie de arquivo geológico do início do Sistema Solar, sem vento para apagar marcas nem água para redesenhar relevos.
